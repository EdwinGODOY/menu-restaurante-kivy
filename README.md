Menú de Restaurante 

Aplicación desarrollada en Python usando Kivy.
Muestra un menú de restaurante con categorías y productos,
cargados desde un archivo JSON 

Funcionalidades
- Pantalla inicial con nombre del restaurante
- Pantalla de categorías
- Pantalla de productos por categoría
- Precios visibles
- Datos cargados desde archivo JSON

Tecnologías
- Python 3
- Kivy

Ejecución del proyecto
1. Crear entorno virtual
   source env/Scripts/activat
3. Instalar dependencias:
   pip install kivy
4. Ejecutar:
   python main.py

Ejecutable
El archivo ejecutable (.exe) se encuentra en la sección **Releases**
del repositorio.
